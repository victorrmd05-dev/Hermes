# 🧠 Skill_Anthropic_Global_Suite_V5

## Visão Geral
Esta documentação consolida a suite global de skills desenvolvida pela Anthropic, projetada para expandir as capacidades do Nexus em criação de documentos, testes de aplicações web, geração de arte e muito mais. Estas ferramentas oferecem um arsenal robusto para desenvolvimento de código, criação de conteúdo e automação.

## 🛠️ Ferramentas Globais Instaladas e Comandos Rápidos

- **skill-creator**: Criação, edição, teste e otimização de skills.
  - *Comando Rápido*: "Use o skill-creator para criar/melhorar a skill [nome_da_skill]."
- **webapp-testing**: Interação e testes automáticos de aplicações web locais usando Playwright.
  - *Comando Rápido*: "Use o webapp-testing para testar a interface na url [URL]."
- **claude-api**: Construção, debug e otimização de apps usando Claude API e Anthropic SDK (incluindo prompt caching).
  - *Comando Rápido*: "Use o claude-api para refatorar esta chamada de API do Anthropic SDK."
- **mcp-builder**: Guias para criação de servidores MCP (Model Context Protocol).
  - *Comando Rápido*: "Use o mcp-builder para criar um MCP server de integração com [serviço]."
- **web-artifacts-builder**: Criação de artefatos complexos com React, Tailwind CSS e shadcn/ui.
  - *Comando Rápido*: "Use o web-artifacts-builder para construir um componente de dashboard."
- **frontend-design**: Geração de interfaces web com qualidade de produção e design premium.
  - *Comando Rápido*: "Use o frontend-design para estilizar e criar esta landing page."
- **canvas-design** / **algorithmic-art**: Criação de artes visuais, documentos estáticos e design algorítmico (p5.js).
  - *Comando Rápido*: "Use o canvas-design para gerar a arte visual do projeto."
- **docx**, **pdf**, **pptx**, **xlsx**: Manipulação de documentos do Word, PDFs, apresentações e planilhas Excel (leitura, edição e formatação avançada).
  - *Comando Rápido*: "Use a skill [formato] para extrair os dados deste arquivo."
- **doc-coauthoring**: Workflow estruturado para co-autoria de propostas, specs técnicas e RFCs.
  - *Comando Rápido*: "Vamos usar o doc-coauthoring para rascunhar a especificação técnica."
- **internal-comms** / **slack-gif-creator**: Comunicações internas (status reports) e criação de GIFs para Slack.
  - *Comando Rápido*: "Use o internal-comms para formatar o memorando semanal."
- **brand-guidelines** / **theme-factory**: Aplicação de cores e tipografia da marca, ou temas consistentes.
  - *Comando Rápido*: "Aplique a brand-guidelines neste documento gerado."

## 🚀 Metodologia de Uso
Para utilizar qualquer uma das ferramentas desta suite, você não precisa necessariamente chamar o nome exato do arquivo; o Nexus acionará automaticamente a skill adequada baseada na sua intenção (como "quero modificar um PDF", ou "crie uma nova skill de automação"). Porém, utilizar os nomes exatos garante a execução precisa da ferramenta correta.
