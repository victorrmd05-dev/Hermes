# 📊 Dashboard de Tráfego Pago (Facebook Ads)

**Contexto:**
Dashboard para monitoramento e análise de métricas do Facebook Ads, integrado via Supabase ao projeto React.

**Tecnologias:**
- Frontend: React + TypeScript (Next.js 14)
- Backend/DB: Supabase
- Integração: API do Facebook Graph / API do Meta Ads (via Token real)
- Motor de Inteligência: IA Generativa baseada em Doutrina Meta Ads Sênior

**Arquitetura:**
- Veja o Canvas: [[Workflow_Ads_FB.canvas]]

## 🚀 Implementações Recentes (Maio/2026)
- **Funil de Conversão Premium:** Implementação de um funil visual de alta performance com trapézios dinâmicos e métricas de transição.
- **Summary Header:** Nova seção de topo com cards consolidados (Investimento, CPA, ROAS, Faturamento) com indicadores de tendência (+/- %).
- **Lógica de Agregação:** Sistema de cálculo automático para consolidar métricas de todas as campanhas ativas em uma visão macro no topo do dashboard.
- **Design Elite:** UI atualizada com estilo glassmorphism, tipografia otimizada e foco total em legibilidade e contraste para tomada de decisão rápida.
- **Integração Claude Ads Health Widget:** Incorporação de um painel de monitoramento de IA dinâmico diretamente na UI, mostrando a pontuação de saúde das campanhas e status de sub-agentes online (*Copywriter IA, Visual Designer, Ads Strategist, Audit Engine*).
- **Core Motor IA Sênior (`ads-meta-senior`):** Nova inteligência analítica baseada no framework matemático de elite **80x10x10** (Connect Rate $\ge$ 80% → Checkout Rate $\ge$ 10% → Purchase Rate $\ge$ 10%), com regras duras para escala (vertical e horizontal) e árvore de diagnósticos ágeis.
- **Estruturação VPS-Ready (Pronto para VPS):** Cópia de todo o ecossistema de sub-agentes, scripts Python (geração de relatórios PDF, auditoria avançada) e referências de mercado diretamente na raiz do projeto (`src/lib/ai-agents/`), garantindo execução offline e autônoma após o deploy em produção.
